# VoxCity - Sistema de Denúncias Urbanas

Bem-vindo ao repositório do **VoxCity**, o megafone digital da nossa cidade!

## 🛠️ Pré-requisitos

Antes de começar, você precisa ter instalado na sua máquina:
* [Git](https://git-scm.com)
* [Node.js](https://nodejs.org/) (Versão LTS recomendada)

## 📥 Como baixar o projeto

No seu terminal, rode o comando abaixo para clonar o repositório:
git clone https://github.com/Vitoriokaua/voxcity.git

Depois, entre na pasta do projeto:
cd voxcity

## 🖥️ Como rodar o Frontend (React)

Abra um terminal, certifique-se de estar na pasta raiz do projeto e siga os passos:

1. Entre na pasta do frontend: 
   cd frontend
2. Instale todas as dependências do React: 
   npm install
3. Inicie o servidor de desenvolvimento: 
   npm start

O site vai abrir automaticamente no seu navegador (geralmente no endereço http://localhost:3000).

## ⚙️ Como rodar o Backend (Node.js)

*Abra um terminal, certifique-se de estar na pasta raiz do projeto e siga os passos:

1. Entre na pasta do backend:
   cd backend
2. Instale todas as dependências e ferramentas do servidor:
   npm install
3. Gere os arquivos do cliente do Prisma:
   npx prisma generate
4. Inicie o servidor de desenvolvimento:
   npm run dev*
